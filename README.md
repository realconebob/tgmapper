# TGMapper

## A Tool for Analyzing the Connections Between Telegram Channels

TGMapper was born out of a curiosity to explore how Telegram channels interact with each other. Telegram offers features like post forwarding, which much like Twitter's retweet function, lets you repost one channel's post to another, with a link back to the original. In practice, channels that engage in lots of forwarding create small communities of like-minded users. I've seen this happen several times, and I want to see just how far the forwarding rabbit hole goes
